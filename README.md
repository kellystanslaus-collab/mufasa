# XIVIA — Social platform

Starter scaffold for the XIVIA social platform (Next.js + TypeScript + Supabase).

## Features
- Next.js + TypeScript frontend
- Supabase client dependency included
- Basic layout and API placeholder

## Local development
1. npm install
2. Copy `.env.example` to `.env.local` and set your SUPABASE_URL and SUPABASE_ANON_KEY
3. npm run dev

## Deployment
- Deploy to Vercel for quick hosting. Connect the repo and set environment variables in the dashboard.
